# About + Tagline de Page — Plantilla con SEO (MODO B)

> Tagline 120c · About 2.000c. Front-load keywords que tu buyer busca. El About es indexable (búsqueda interna + Google).

## Tagline (120c)
`<<propuesta de valor + categoría + keyword>>`  — (`__/120`)
Ej: `Innovation & Creative Technology Agency · GEO y visibilidad de marca en IA generativa`

## About (2.000c) — estructura
| Bloque | Contenido |
|--------|-----------|
| 1. Qué hacéis + para quién (keyword-first) | `<<>>` |
| 2. Problema que resolvéis / categoría | `<<>>` |
| 3. Prueba (clientes, cifras, casos) | `<<>>` |
| 4. Servicios / líneas (specialties como keywords) | `<<>>` |
| 5. CTA (contacto, web, "Síguenos") | `<<>>` |

```
<<Frase 1: qué sois y para quién, con la keyword categórica al frente>>

<<Problema de mercado + cómo lo resolvéis>>.

<<Prueba: nº clientes, marcas líderes, resultados agregados>>.

Qué hacemos: <<servicio 1>>, <<servicio 2>>, <<servicio 3>>.

<<CTA + web + invitación a seguir la página>>.
```

## Specialties (máx 20, son keywords de búsqueda)
`<<kw1>>, <<kw2>>, <<kw3>>, … <<kw20>>`

## CTA de página
`<<Visitar sitio web / Contactar / Registrarse / Más información>>`

## Hashtags de comunidad (máx 3, camelCase)
`#<<Marca>>` `#<<Categoria>>` `#<<Tema>>`
